Пояснение:

    HTMLFetcher.cs: Универсальная библиотека для получения HTML-кода сайта.
    MainWindow.xaml: Главное окно, содержащее TextBox для ввода URL и кнопку для загрузки.
    MainWindow.xaml.cs: Обработчик нажатия кнопки, который использует библиотеку для получения HTML и открывает новое окно с результатами.
    HtmlUtilities.cs: Утилиты для извлечения чистого текста и заголовка из HTML-кода.
    ResultWindow.xaml: Окно для отображения результатов, заголовок которого устанавливается в заголовок сайта, а тело содержит чистый текст.
